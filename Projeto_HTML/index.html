História da Web

Voltar ao topo

Antes de começarmos a estudar HTML é extremamente importante sabermos onde tudo começou para podermos entender como a internet funciona atualmente.

Então, voltaremos no ano de 1944 para conhecermos os primeiros computadores.

Havard Mark I

O MARK I foi um computador totalmente eletromecânico construído em 1944 pelo professor Howard Aiken da Universidade de Harvard nos Estados Unidos. Com cerca de 17 metros de comprimento por 2,5 metros de altura e uma massa de cerca de 5 toneladas o MARK I foi construído durante a Segunda Guerra Mundial visando ser um calculador de tabelas para uso na navegação.

Colossus

O Colossus foi um computador criado para criptoanálises pela Alemanha em 1946 durante a Segunda Guerra Mundial.

ENIAC

Criado pelos Estados Unidos, o ENIAC também foi um computador criado para criptoanálises em 1946 durante a Segunda Guerra Mundial.

Segunda Geração de Computadores

A segunda geração de computadores chegaram no ano de 1959 e durou até 1965, trazendo com eles a evolução das válvulas para transistores e uma considerável diminuição de tamanho.

Terceira geração de computadores

A terceira geração de computadores chegaram no ano de 1965 e durou até 1970, trazendo com eles a evolução dos transistores para os circuitos integrados trazendo mais desempenho e um menor tamanho.

Quarta geração de Computadores

A terceira geração de computadores chegaram no ano de 1971 e dura até hoje e continua evoluindo até hoje.

Surgimento da Web

Joseph Carl Robnette Licklider criou uma teoria sobre uma rede onde todos poderiam se comunicar através dela.

Apos muito tempo de estudo surgiu a Arpanet, um sistema de pacotes que envia um de cada vez a vários destinos.

Com a evolução dos anos, Viton Cerf e Robert Kahn criaram o TCP/IP, um conjunto de protocolos de comunicação entre computadores em rede.

Em 1989, Tim Berners-Lee criou a famosa World Wide Web (WWW) a comunicação hipertextos.

Grandes Nomes

Grace Murray Hopper

Da marinha dos Estados Unidos, Grace Murray Hopper foi a primeira programadora do MARK I, criadora da linguagem de programação de alto nível Flow-Matic (Base do COBOL) e também popularizou a expressão de bug.

Joseph Carl Robnett Licklider

recrutado pela DARPA por teorizar sobre uma rede “galática” de computadores.

Robert Kahn

Um dos criadores do TCP/IP e tambem apresentador do primeiro e-mail.

Tim Berners-Lee

Criador do World Wide Web, Tim Berners-Lee também popularizou o HTTP E HTML.

Marc Andreessen

Inventor reinventor do Netscape feito a partir do mosaico, deteve 90% da Internet, superado apenas pelo Internet Explorer anos mais tarde.

Bill Gates

Criador da gigantesca empresa Microsoft que é criadora do Windows e Internet Explorer.

A chave do sucesso nos negócios, perceber aonde o mundo vai e chegar ali primeiro.

Steve Jobs

Fundador da Apple que popularizou os smartphones que conhecemos hoje com o lançamento do iPhone.

O que são Servers

Voltar ao topo

Servidores

Um servidor web é um computador que armazena arquivos que compõem os sites.

Hardware

O hardware(componentes físicos) para servidores geralmente precisam ser mais robustos para poder suportar vários cálculos e acessos de uma única vez.

Software

Software para servidores são programas usados para executar tarefas de um servidor.

Tipos de Servidores

Servidor Proxy serve para filtrar sites acessados.

Hospedagem de sites

Hospedagem de sites é quando você instala seu site em um servidor.

Linguagens de Programação

Voltar ao topo

Linguagem de Programação é uma maneira de escrever códigos de computadores que se transformam em códigos binários (0 e 1) para que um computador possa entender e executar uma função.

Server-Side

Quando falamos em linguagens Server-Side nos referimos a linguagens voltadas para o servidor como, por exemplo, php.

Client-Side

Quando falamos em linguagens Client-Side nos referimos a linguagens voltadas para o Cliente.

HTML

Algo importante a ser lembrado é que HTML não é uma linguagem de programação e sim uma forma de marcação/edição de textos para que o navegador possa entender e organizar o conteúdo.

Estrutura HTML

A estrutura do HTML consiste em duas partes que são Head e body.

Head é tudo que será carregado antes de o site aparecer para o usuário.

Body é tudo que o usuário consegue ver no site.
